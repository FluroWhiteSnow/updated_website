PROJECTS

Two-Way Marketplace

This was my second project I completed. This is a two way marketpalce that connect buyers and sellers together. Users are able to create their own account, and list and buy items. Users are also able to message other users, and have a persistent log of messages. Messenging is however not live due to time restraints on the project. To see new messeges you have to refresh the page.
I integrated the Stripe API to allow for purchasing functionality. UltaHook was used as a webhook to allow for end-to-end TLS and persistent endpoints. This application was built with Ruby, on the Ruby on Rails framework. I also integrated CRUD operations through PostgreSQL.

Technologies used:

  • Ruby
  • Ruby on Rails
  • PostgreSQL
  • Stripe API
  • AWS S3
  • Heroku

Recipe-Book Terminal App

My first small coding project in terminal! In this application you are apple to log in, browse built in recipes and create, update and delete recipe your own recipes. The recipes you save to your use are only viewable by you and no one else. If you wanted to submit your recipe to the global recipe, you can submit your recipe for review.
The user "admin" will then be able to review the recipe you submitted, and then if they like it they can accept the request to submit the recipe to the global recipes book. On next log in for users, the recipe will automatically be added to everyones books. You are allowed to edit built in recipes, however these are saved as a new recipe, and you cannot delete built in recipes.

Technologies used:

  • Ruby
  • YAML
  • Git
